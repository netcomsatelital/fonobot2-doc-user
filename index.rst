===========================
Guía del usuario de Fonobot
===========================

Esta guía cubre la documentación de usuario de la aplicación *Fonobot*.

Fonobot es un sistema que realiza llamados telefónicos de manera automática,
esto es, un *robot llamador*.  Está implementado como un servicio web y
funciona completamente desde la nube, de manera que no hay necesidad de
instalar ni equipamientos ni software para utilizarlo.  Los únicos requisitos
para administrar una cuenta son una conexión a internet y un navegador web.

Típicamente el sistema se usa para llevar a cabo campañas publicitarias
masivas, de manera similar a un anuncio por radio o televisión.
